About

Este proyecto es un sistema OpenSource de Chessbot.
Que esta separado en dos repositorios,
el engine de ajedrez y el sistema de control del brazo

Sigue Leyendo

Requerimientos

Para poder correr el Engine de ajedrez necesitas tener instalado en tu computador una version igual o superior a la 7 de java develoment kit.
En el siguiente link se anexa un tutorial para poder instalarlo.

Como lo descargo?

Una vez que tienes todos los requerimientos instalados necesitas descargar el archivo binario del engine para descargarlo dale al boton de Download de la parte de arriba.

Ejecutandolo

Una vez que tienes descargado el binario del engine lo que necesitas hacer para ejecutarlo es abrir la shell de tu sistema operativo, y situarte en la carpeta donde se descargo el binario y ejecuta el siguiente comando.

java -jar chess.jar
